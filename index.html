Lesson 1: Basics of the Internet

How the web works

The internet breaks down into a few basic elements: HTML, URLs, HTTP, and Web Apps. The World Wide Web, as we know it, is a collection of HTML Documents. HTML stands for Hyper Text Markup Language.Hyperlinks are also used to connect one web page to another. Here is a nice breakdown.

The major pieces of the Web are You (browser/comp) <=> The Internet (drawn as a cloud per industry standard) <=>Servers (Internet uses HTTP to talk to servers).

  • HTML files are the structure. Where the walls are in the house
  • CSS are the style of the site. Paint, rugs, curtains.
  • Java Script is how you change/interact with components. Remote, garage door opener.

The Parts of HTML

  • Text content: what you see
  • Markup: describes the document
  • Reference to other documents: images, videos.
  • Links to other pages: hyperlinks

The HTML Process

Plain Text: you get what you type
Markup: make the text look different

Markup is made up of TAGs.

Opening tag <name> Contents </name> Closing tag

*if you do not close the tag, your opening tag attributes will continue on.
<name> contents </name> This whole unit is often referred to as an element.

Bold Tags <b> contents </b>
Emphasis Tags/Italics Tags (EM Tags) <em> contents </em>

HTML Attributes

TAGs can have multiple attributes. Attributes have a name (ATTR) and then = a “value”. An example of a TAG which uses an attribute is the Anchor TAG. Anchor TAGs are for making links.

Anchor Tag: <a "value" > plain text contents </a >

Example: <a href(this is the attr. name) = “www.reddit.com” (this is the value) > derp(this is the contents) </a>

If you were to render this in HTML, you would just see the word derp and it would be a link to www.Reddit.com.

Image TAG

<img> is for images
<img src=“url” alt=“text”>

src is the source attribute. “url" is the location of the image to download. alt is the attribute for alternate and the “text” that goes with it is the message that is displayed when the image does not load. It is also what is displayed for the blind. Nothing will break, but you should include alt=“text”

*Image TAGs do not have a contents section and therefore no closing TAG.

* These types of TAGs are called Void TAGs

Whitespace

Writing in HTML will deliver all text in the same line without any spaces between lines of text. To create space you should use the break TAG < br >. < br > is a void TAG (no closing TAG). Use < br > after text where you want a break.

Break Example:

1 This text is really
2 < br >
3 < br >
4 too long for one line.

Another way of doing line breaks is to use the Paragraph TAG. <p> The <p> uses content and a closing TAG. <p> content </p>
* The <p> TAG will create an invisible box around the content.
Inline/Block
<br> TAGs are inline TAGs. They make text wrap to the next line.
<p> TAGs are Block TAGs and they put the text in an invisible box which has hieght and width, and will give space between the content in the <p> TAG and the rest of the text.
Span/Div
<span> content </span> is Inline.
<div> content </div> is Block.

These elements only contain their content. There is a way to attach styles to them to adjust the behaviors of how they display.

Example: you could use a CSS class to adjust either.
<span class=“foo”> content </span>
Side examples:
<strong> Bolds Content </strong> Inline TAGs
<form> Moves content to a new line and creates space at the bottom </form> Block TAGs

Lesson 3: Working with CSS

CSS

*Avoid repetition. This is a key to using CSS and a fundamental to programming.
CSS stands for Cascading Style Sheets. It is this Cascading effect that gives CSS it's power to streamline your work.

If you want to write CSS that makes all h1 elements have a black background and white text, you would write:

h1 { background-color : black; color: white; }

In this example, the h1 is a selector. It says to the browser "I want you to apply the rules I'm about to tell you to every h1 element." After the h1 there is a left curly brace { and at the bottom there is a matching right curly brace }. Everything between these curly braces will be interpreted as a "rule" that should be applied to every h1 element. The line of code that says background-color : black; is a declaration.background-color is a property and black is the value of that property.

Class selectors allow you to add style to certain elements. (only certain h1 tags, for instance)

You add class selectors by writing something like this:

  • < div class ="stop" >< /div >
  • < div class="slow" >< /div >
  • < div class="go" >< /div >
Then you can effect those classes you have created in CSS by writing them a .class-name , like below:
  • .stop { background-color: red; }
  • .slow { background-color: yellow; }
  • .go { background-color: green; }
* Rules can be applied to all divs or just certain divs by using the appropriate selector.

The Box Model

Understanding the box model that containers (div/span) create means that you can shape the space that your content inhabits and how it looks in that space.

Each container will have the content, then padding for the content, then a border, and then a margin around the outside that seperates that container from other containers.

In more detail, padding clears and area around the content and is affected by the background color. It is the space inside of the box that protects the content directly. The bordergoes around the padding. it is inherited from the color property of the box. The margin is the space between boxes. It has no background color and is transparent. You can set the size of the box by pixels or percentage. Pixels will not change based on the size of the screen and this could effect user experience. *This is especially true for mobile apps. You can also set a max width.

*It’s important to know that the actual content witdth involves the content, border and padding widths.

This is how you effect box sizes in CSS

*{ box-sizing: border-box; }

Older browsers need browser specific prefixes to display.

  • webkit-box-sizing: border-box;
  • moz-box-sizing: border-box;
  • ms-box-sizing: border-box;

Flex Box Layout: An efficient way to layout a line or elements in a container.

If you want several elements to be next to each other, you can change the display attr. of the parent container to value x.

Ex: < div class=“app” style” display: flex; “ >

This makes all divs in the class of app fall in line with each other. (For it to work, you also have to give the elements a size smaller than the default 100%. Otherwise there is not enough room. This is when you would use the border sizing.) Here is a link to an online guide.
